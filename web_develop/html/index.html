有機農法で作られた植物を原料として、作られた、こだわりのオリジナルソープです。
もっちりとした泡立ちで、お肌のうるおいを守りながら心地よく洗い上げます。敏感肌の方にもおすすめです。
成分
ココナッツオイル、オリーブオイル、植物性グリセリン、シアバター、アルガンオイル、天然香科 注意事項
・開封後はなるべく早めにお使いください。
・湿気に弱いため、使用後は水気をよく切り、通気性の良い場所に保管してください。
・乳幼児の手が届かない場所においてください。
・目に入ったときは、必ずすぐに流水で充分に洗い流してください。
・肌に合わないときは、使用を中止し相談してください。
・効果・効能については、個人差が有ります。
